# Référence locale (optionnelle)

Ce dossier peut contenir un **snapshot de code ou de schéma** utilisé uniquement en développement pour accélérer la conception des modules.

- **Non requis** au build ni au déploiement de SaaS OS.  
- Pour un dépôt **public** : le sous-dossier `_vendor_snapshot/` est en **gitignore** — chaque équipe y place sa copie locale si besoin.

SaaS OS reste une **installation neuve par instance** (voir README et `docs/INSTALL.md`).
